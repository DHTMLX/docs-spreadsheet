# 데이터 검색 {#searching-for-data}

검색 바를 열려면 다음 두 가지 방법 중 하나를 사용합니다:

- 임의의 셀에 포커스를 설정하고 **Ctrl (Cmd) + F**를 누릅니다
- 또는 *데이터* -> *검색*으로 이동합니다
