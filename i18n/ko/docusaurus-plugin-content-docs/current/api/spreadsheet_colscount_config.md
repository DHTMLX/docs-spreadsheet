# colsCount

### 설명 {#description}

@short: 선택 사항. 초기화 시 스프레드시트의 열 수를 설정합니다

### 사용법 {#usage}

~~~jsx
colsCount?: number;
~~~

### 예제 {#example}

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
    colsCount: 10,
    // 기타 설정 매개변수
});
~~~
