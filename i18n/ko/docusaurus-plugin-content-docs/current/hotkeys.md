# 단축키 목록 {#hot-keys-list}

다음은 DHTMLX Spreadsheet 사용 시 활용할 수 있는 단축키 목록입니다.

## 편집 {#editing}

<table>
    <tbody>
        <tr>
            <td><b>Enter</b></td>
            <td>셀의 편집기를 활성화합니다</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Enter</b></td>
            <td>셀 편집을 중지하고 해당 셀에 선택 상태를 유지합니다</td>
        </tr>
        <tr>
            <td><b>F2</b></td>
            <td>선택된 셀에서 편집기를 엽니다</td>
        </tr>
        <tr>
            <td><b>Esc</b></td>
            <td>변경 사항을 저장하지 않고 셀 편집기를 닫습니다</td>
        </tr>
    </tbody>
</table>

## 탐색 {#navigation}

{{note 탐색은 셀이 편집 중이지 않은 경우에만 작동합니다.}}

<table>
    <tbody>
        <tr>
            <td><b>Enter</b></td>
            <td>셀의 편집기를 활성화합니다. 편집기를 닫고 선택을 한 셀 아래로 이동합니다(열 방향). 셀 범위가 선택된 경우 범위 내에서 한 셀 아래로 이동합니다.</td>
        </tr>
        <tr>
            <td><b>Shift + Enter</b></td>
            <td>선택을 한 셀 위로 이동합니다(열 방향). 셀 범위가 선택된 경우 범위 내에서 한 셀 위로 이동합니다. 열의 첫 번째 셀에 도달하면 비활성화됩니다.</td>
        </tr>
        <tr>
            <td><b>Tab</b></td>
            <td>선택을 한 셀 오른쪽으로 이동합니다(행 방향). 셀 범위가 선택된 경우 범위 내에서 한 셀 오른쪽으로 이동합니다. 행의 가장 오른쪽 셀에 도달하면 비활성화됩니다.</td>
        </tr>
        <tr>
            <td><b>Tab + Shift</b></td>
            <td>선택을 한 셀 왼쪽으로 이동합니다(행 방향). 셀 범위가 선택된 경우 범위 내에서 한 셀 왼쪽으로 이동합니다. 행의 가장 왼쪽 셀에 도달하면 비활성화됩니다.</td>
        </tr>
        <tr>
            <td><b>화살표 키</b></td>
            <td>시트의 셀 간 탐색을 제공합니다</td>
        </tr>
        <tr>
            <td><b>Home</b></td>
            <td>선택을 행의 첫 번째 셀로 이동합니다</td>
        </tr>
        <tr>
            <td><b>End</b></td>
            <td>선택을 행의 마지막 셀로 이동합니다</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 왼쪽 화살표 키</b></td>
            <td>선택을 행의 첫 번째 셀로 이동합니다</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 오른쪽 화살표 키</b></td>
            <td>선택을 행의 마지막 셀로 이동합니다</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 위쪽 화살표 키</b></td>
            <td>선택을 열의 첫 번째 셀로 이동합니다</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 아래쪽 화살표 키</b></td>
            <td>선택을 열의 마지막 셀로 이동합니다</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Home</b></td>
            <td>선택을 시트의 시작 부분(왼쪽 상단 셀)으로 이동합니다</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + End</b></td>
            <td>선택을 시트의 끝 부분(오른쪽 하단 셀)으로 이동합니다</td>
        </tr>
        <tr>
            <td><b>Shift + 위쪽/아래쪽/왼쪽/오른쪽 화살표 키</b></td>
            <td>선택된 셀에서 선택 영역을 확장합니다</td>
        </tr>
        <tr>
            <td><b>Page Up</b></td>
            <td>시트를 한 화면 위로 스크롤합니다</td>
        </tr>
        <tr>
            <td><b>Page Down</b></td>
            <td>시트를 한 화면 아래로 스크롤합니다</td>
        </tr>
    </tbody>
</table>
