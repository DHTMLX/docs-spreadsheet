# Список горячих клавиш {#hot-keys-list}

Ниже представлен список горячих клавиш, которые можно использовать при работе с DHTMLX Spreadsheet.

## Редактирование {#editing}

<table>
    <tbody>
        <tr>
            <td><b>Enter</b></td>
            <td>активирует редактор в ячейке</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Enter</b></td>
            <td>завершает редактирование ячейки, оставляя выделение на ней</td>
        </tr>
        <tr>
            <td><b>F2</b></td>
            <td>открывает редактор в выбранной ячейке</td>
        </tr>
        <tr>
            <td><b>Esc</b></td>
            <td>закрывает редактор ячейки без сохранения изменений</td>
        </tr>
    </tbody>
</table>

## Навигация {#navigation}

{{note Навигация работает только в том случае, если ни одна ячейка не находится в режиме редактирования.}}

<table>
    <tbody>
        <tr>
            <td><b>Enter</b></td>
            <td>активирует редактор в ячейке. Закрывает редактор и перемещает выделение на одну ячейку вниз (в столбце). Если выбран диапазон ячеек, перемещает выделение на одну ячейку вниз в пределах диапазона.</td>
        </tr>
        <tr>
            <td><b>Shift + Enter</b></td>
            <td>перемещает выделение на одну ячейку вверх (в столбце). Если выбран диапазон ячеек, перемещает выделение на одну ячейку вверх в пределах диапазона. Деактивируется при достижении первой ячейки в столбце.</td>
        </tr>
        <tr>
            <td><b>Tab</b></td>
            <td>перемещает выделение на одну ячейку вправо (в строке). Если выбран диапазон ячеек, перемещает выделение на одну ячейку вправо в пределах диапазона. Деактивируется при достижении крайней правой ячейки в строке.</td>
        </tr>
        <tr>
            <td><b>Tab + Shift</b></td>
            <td>перемещает выделение на одну ячейку влево (в строке). Если выбран диапазон ячеек, перемещает выделение на одну ячейку влево в пределах диапазона. Деактивируется при достижении крайней левой ячейки в строке.</td>
        </tr>
        <tr>
            <td><b>Клавиши со стрелками</b></td>
            <td>обеспечивают навигацию по ячейкам листа</td>
        </tr>
        <tr>
            <td><b>Home</b></td>
            <td>перемещает выделение к первой ячейке в строке</td>
        </tr>
        <tr>
            <td><b>End</b></td>
            <td>перемещает выделение к последней ячейке в строке</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + стрелка влево</b></td>
            <td>перемещает выделение к первой ячейке в строке</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + стрелка вправо</b></td>
            <td>перемещает выделение к последней ячейке в строке</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + стрелка вверх</b></td>
            <td>перемещает выделение к первой ячейке в столбце</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + стрелка вниз</b></td>
            <td>перемещает выделение к последней ячейке в столбце</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Home</b></td>
            <td>перемещает выделение к началу листа (верхняя левая ячейка)</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + End</b></td>
            <td>перемещает выделение к концу листа (нижняя правая ячейка)</td>
        </tr>
        <tr>
            <td><b>Shift + стрелка вверх/вниз/влево/вправо</b></td>
            <td>расширяет выделение от выбранной ячейки</td>
        </tr>
        <tr>
            <td><b>Page Up</b></td>
            <td>прокручивает лист на один экран вверх</td>
        </tr>
        <tr>
            <td><b>Page Down</b></td>
            <td>прокручивает лист на один экран вниз</td>
        </tr>
    </tbody>
</table>

## Поиск {#search}

<table>
    <tbody>
        <tr>
            <td><b>Ctrl (Cmd) + F</b></td>
            <td>открывает строку поиска в правом верхнем углу таблицы</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + G</b></td>
            <td>перемещает выделение к следующему найденному совпадению</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Shift + G</b></td>
            <td>перемещает выделение к предыдущему найденному совпадению</td>
        </tr>
    </tbody>
</table>

## Выделение {#selection}

<table>
    <tbody>
        <tr>
            <td><b>Ctrl (Cmd) + A</b></td>
            <td>выделяет все ячейки на листе</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + щелчок левой кнопкой мыши</b></td>
            <td>позволяет выбирать несколько несмежных ячеек</td>
        </tr>
        <tr>
            <td><b>Shift + щелчок левой кнопкой мыши</b></td>
            <td>позволяет выбрать диапазон ячеек, где выбранная ячейка является первой<br> ячейкой диапазона, а кликнутая — последней</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Shift + щелчок левой кнопкой мыши</b></td>
            <td>позволяет выбирать несколько несмежных диапазонов ячеек</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Space</b></td>
            <td>выделяет весь столбец</td>
        </tr>
        <tr>
            <td><b>Shift + Space</b></td>
            <td>выделяет всю строку</td>
        </tr>
    </tbody>
</table>

<table>
    <tbody>
