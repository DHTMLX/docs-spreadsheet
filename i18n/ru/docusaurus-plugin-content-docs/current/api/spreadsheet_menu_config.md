# menu

### Описание {#description}

@short: Необязательный. Показывает/скрывает меню

### Использование {#usage}

~~~jsx
menu?: boolean;
~~~

### Конфигурация по умолчанию {#default-config}

~~~jsx
menu: true
~~~

### Пример {#example}

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
  menu: false,
  // other config parameters
});
~~~
