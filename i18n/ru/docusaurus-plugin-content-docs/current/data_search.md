# Поиск данных {#searching-for-data}

Чтобы открыть панель поиска, воспользуйтесь одним из двух способов:

- Установите фокус на любую ячейку и нажмите **Ctrl (Cmd) + F**
- Или перейдите в меню: *Data* -> *Search*
