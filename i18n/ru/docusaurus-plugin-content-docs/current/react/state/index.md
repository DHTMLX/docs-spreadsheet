## Ключевая концепция {#key-concept}

Проп `sheets` является **единственным источником истины** для всех данных таблицы. Передайте массив объектов [`SheetData`](react/types.md#sheetdata), и обёртка сравнит ваши данные с текущим состоянием виджета, применив только изменения. Используйте иммутабельные обновления (оператор расширения, функциональные обновления `setState`), чтобы React мог эффективно обнаруживать изменения.
