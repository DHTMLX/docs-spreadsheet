# 快捷键列表 {#hot-keys-list}

以下是在使用 DHTMLX Spreadsheet 时可以使用的快捷键列表。

## 编辑 {#editing}

<table>
    <tbody>
        <tr>
            <td><b>Enter</b></td>
            <td>激活单元格中的编辑器</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Enter</b></td>
            <td>停止单元格编辑，保持选中该单元格</td>
        </tr>
        <tr>
            <td><b>F2</b></td>
            <td>在选中的单元格中打开编辑器</td>
        </tr>
        <tr>
            <td><b>Esc</b></td>
            <td>关闭单元格中的编辑器，不保存更改</td>
        </tr>
    </tbody>
</table>

## 导航 {#navigation}

{{note 导航仅在没有单元格处于编辑状态时有效。}}

<table>
    <tbody>
        <tr>
            <td><b>Enter</b></td>
            <td>激活单元格中的编辑器。关闭编辑器并将选中项向下移动一个单元格（在列中）。如果选中了单元格区域，则在区域范围内向下移动一个单元格。</td>
        </tr>
        <tr>
            <td><b>Shift + Enter</b></td>
            <td>将选中项向上移动一个单元格（在列中）。如果选中了单元格区域，则在区域范围内向上移动一个单元格。到达列的第一个单元格时停止。</td>
        </tr>
        <tr>
            <td><b>Tab</b></td>
            <td>将选中项向右移动一个单元格（在行中）。如果选中了单元格区域，则在区域范围内向右移动一个单元格。到达行的最右侧单元格时停止。</td>
        </tr>
        <tr>
            <td><b>Tab + Shift</b></td>
            <td>将选中项向左移动一个单元格（在行中）。如果选中了单元格区域，则在区域范围内向左移动一个单元格。到达行的最左侧单元格时停止。</td>
        </tr>
        <tr>
            <td><b>方向键</b></td>
            <td>在工作表的单元格之间导航</td>
        </tr>
        <tr>
            <td><b>Home</b></td>
            <td>将选中项移至行中的第一个单元格</td>
        </tr>
        <tr>
            <td><b>End</b></td>
            <td>将选中项移至行中的最后一个单元格</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 左方向键</b></td>
            <td>将选中项移至行中的第一个单元格</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 右方向键</b></td>
            <td>将选中项移至行中的最后一个单元格</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 上方向键</b></td>
            <td>将选中项移至列中的第一个单元格</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 下方向键</b></td>
            <td>将选中项移至列中的最后一个单元格</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Home</b></td>
            <td>将选中项移至工作表的起始位置（左上角单元格）</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + End</b></td>
            <td>将选中项移至工作表的末尾位置（右下角单元格）</td>
        </tr>
        <tr>
            <td><b>Shift + 上/下/左/右方向键</b></td>
            <td>从选中的单元格扩展选区</td>
        </tr>
        <tr>
            <td><b>Page Up</b></td>
            <td>向上滚动工作表一屏</td>
        </tr>
        <tr>
            <td><b>Page Down</b></td>
            <td>向下滚动工作表一屏</td>
        </tr>
    </tbody>
</table>

## 选择 {#selection}

<table>
    <tbody>
        <tr>
            <td><b>Ctrl (Cmd) + A</b></td>
            <td>选中工作表中的所有单元格</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + 鼠标左键单击</b></td>
            <td>允许选中多个不连续的单元格</td>
        </tr>
        <tr>
            <td><b>Shift + 鼠标左键单击</b></td>
            <td>允许选中单元格区域，其中选中的单元格为区域的第一个单元格，单击的单元格为区域的最后一个单元格</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Shift + 鼠标左键单击</b></td>
            <td>允许选中多个不连续的单元格区域</td>
        </tr>
        <tr>
            <td><b>Ctrl (Cmd) + Space</b></td>
            <td>选中整列</td>
        </tr>
        <tr>
            <td><b>Shift + Space</b></td>
            <td>选中整行</td>
        </tr>
    </tbody>
</table>
