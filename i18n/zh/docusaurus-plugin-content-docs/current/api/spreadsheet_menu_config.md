# menu

### 描述 {#description}

@short: 可选。显示/隐藏菜单栏

### 用法 {#usage}

~~~jsx
menu?: boolean;
~~~

### 默认配置 {#default-config}

~~~jsx
menu: true
~~~

### 示例 {#example}

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
  menu: false,
  // 其他配置参数
});
~~~
