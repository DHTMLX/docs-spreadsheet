# menu

### Beschreibung {#description}

@short: Optional. Blendet das Menü ein oder aus

### Verwendung {#usage}

~~~jsx
menu?: boolean;
~~~

### Standardkonfiguration {#default-config}

~~~jsx
menu: true
~~~

### Beispiel {#example}

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
  menu: false,
  // other config parameters
});
~~~
