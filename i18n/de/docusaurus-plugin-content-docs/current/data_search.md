# Daten suchen {#searching-for-data}

Um die Suchleiste zu öffnen, haben Sie zwei Möglichkeiten:

- Setzen Sie den Fokus auf eine beliebige Zelle und drücken Sie **Ctrl (Cmd) + F**
- Oder navigieren Sie zu: *Daten* -> *Suchen*
