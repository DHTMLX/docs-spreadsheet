# editLine

### Description

@short: Optional. Shows/hides the editing bar

### Usage

~~~jsx
editLine?: boolean;
~~~

### Default config

~~~jsx
editLine: true
~~~

### Example

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
	editLine: true,
    // other config parameters
});
~~~
