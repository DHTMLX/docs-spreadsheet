# menu

### Description

@short: Optional. Shows/hides the menu

### Usage

~~~jsx
menu?: boolean;
~~~

### Default config

~~~jsx
menu: true
~~~

### Example

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
  menu: false,
  // other config parameters
});
~~~
