# rowsCount

### Description

@short: Optional. Sets the number of rows a spreadsheet will have on initialization

### Usage

~~~jsx
rowsCount?: number;
~~~

### Example

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
  rowsCount: 10,
  // other config parameters
});
~~~
