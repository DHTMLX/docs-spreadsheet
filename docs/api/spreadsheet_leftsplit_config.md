# leftSplit

### Description

@short: Optional. Sets the number of columns fixed to the left side of the spreadsheet

### Usage

~~~jsx
leftSplit?: number;
~~~

### Example

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
  leftSplit:3,
  // other config parameters
});
~~~
