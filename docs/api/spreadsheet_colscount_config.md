# colsCount

### Description

@short: Optional. Sets the number of columns a spreadsheet will have on initialization

### Usage

~~~jsx
colsCount?: number;
~~~

### Example

~~~jsx {2}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
	colsCount: 10,
    // other config parameters
});
~~~
