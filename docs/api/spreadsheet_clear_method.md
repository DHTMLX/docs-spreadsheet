# clear()

### Description

@short: clears a spreadsheet

### Usage

~~~jsx
clear(): void;
~~~

### Example

~~~jsx {5}
const spreadsheet = new dhx.Spreadsheet("spreadsheet", {
    // config parameters
});
// clears a spreadsheet
spreadsheet.clear();
~~~
